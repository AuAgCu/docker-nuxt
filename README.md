# 環境構築
1. https://www.docker.com/get-started/からアプリをダウンロード
1. ターミナルを起動して下記コマンド  
  `$ brew install docker`
1. docker-composeをインストール  
  `$ brew install docker-compose`
1. ビルド  
  `$ docker-compose build`
1. npm install  
  `$ docker-compose run --rm nuxt npm install`
1. 起動  
  `$ docker-compose up`
1. ブラウザからlocalhost:9001にアクセス
